BIBLIOTECA DE TAGS HTML

    !DOCTYPE>: Defina o tipo de documento HTML.
  1. html>: Define o início e o fim do documento HTML.
  2. head>: Contém informações sobre o documento, como o título da página, como meta tags e CSS.
  3. title>: Define o título da página que aparece na aba do navegador.
  4. body>: Contém o conteúdo da página, como textos, imagens e links.
  5. h1>a h6>: Define os títulos e subtítulos do conteúdo, em ordem hierárquica.
  6. p>: Defina um parágrafo de texto.
  7. a>: Defina um link para outra página da web ou para outro recurso.
  8. img>: Defina uma imagem na página.
  9. ul>: Define uma lista não ordenada.
  10. ol>: Define uma lista ordenada.
  11. li>: Define cada item de uma lista.
  12. table>: Defina uma tabela.
  13. tr>: Define uma linha na tabela
    14. .
  14. td>: Defina uma célula na tabela.
  15. form>: Defina um formulário de entrada de dados.
  16. input>: Defina um campo de entrada de dados em um formulário.
  17. select>: Define uma lista suspensa de opções em um formulário.
  18. textarea>: Defina uma área de texto em um formulário.
  19. button>: Define um botão em um formulário.
  20. div>: Defina uma seção da página.
  21. span>: Defina uma pequena seção da página.
  22. br>: Insira uma quebra de linha.
  23. hr>: Insira uma linha horizontal.
  24. style>: Defina um bloco de estilos CSS
    25. .
  25. script>: Define um bloco de código JavaScript.
  26. meta>: Define as informações de metadados do documento HTML, como descrição, palavras chave codificação de caracteres.
  27. link>: Defina uma conexão entre um documento HTML e um recurso externo, como uma folha de estilo CSS.
  28. header>: Defina o cabeçalho da página, que geralmente contém o logotipo, título e navegação.
  29. footer>: Define o rodapé da página, que geralmente contém informações de contato, direitos autorais e links para redes sociais.
  30. nav>: Defina a navegação principal da página.
  31. article>: Define um conteúdo autônomo, como um artigo de notícia ou blog.
  32. section>: Defina uma seção da página que contém o conteúdo relacionado.
  33. aside>: Define uma seção da página que contém o conteúdo relacionado, mas não é diretamente parte do conteúdo principal.
  34. time>: Defina uma data ou hora em um formato específico.
  35. video>: Defina um vídeo na página.
  36. audio>: Defina um arquivo de áudio na página.
  37. source>: Defina os diferentes formatos de mídia disponíveis para um elemento de mídia.
  38. iframe>: Define um quadro interno na página que contém outro documento HTML.
  39. progress>: Defina uma barra de progresso para mostrar o status de uma tarefa em andamento.
  40. meter>: Defina uma escala de medida para mostrar a quantidade de um valor específico.
  41. datalist>: Defina uma lista de opções para um campo de entrada de dados.
  42. details>: Defina um elemento com detalhes adicionais que podem ser exibidos ou ocultos.
  43. summary>: Define um resumo do elemento details>.
  44. abbr>: Define uma abreviação.
  45. address>: Defina informações de contato, como endereço e número de telefone.
  46. blockquote>: Define uma citação longa.
  47. cite: Define o título de uma obra citada.
  48. q: Defina uma citação curta.
